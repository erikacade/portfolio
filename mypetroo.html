My Pet Roo

Tools used:

  • HTML
  • CSS
  • Adobe Illustrator

This project was a study in responsive web design and branding/graphic design. The idea for the website was an original concept by me for a level 300 class at ASU, however the images and body copy used are from other sources. The site was intended to be mobile-first, informational, and easy to navigate for users.

After coming up with the initial idea of an informational e-commerce site for kangaroo and wallaby care, I started with creating several wireframes for the landing page and contact form. I sought out feedback from classmates to determine the best heirarchy for information. When starting on the mockups, I created the logo using a kangaroo vector icon I found online, placing it over text to cross the 't' in "My Pet Roo." I chose to code only the home page and the contact page to focus on the responsiveness of a text-heavy page with a graphic as well as the responsiveness of a HTML form.
