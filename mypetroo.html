My Pet Roo

The desktop home page.

Tools used:

  • HTML
  • CSS
  • Adobe Illustrator

This project was a study in responsive web design and branding/graphic design. When deciding my mock brief for this project I wanted to dive into something unconventional but familiar to me. Growing up, my family briefly cared for a young wallaby born in captivity and I always wondered how my mom found information on what to feed him. The site was intended to be mobile-first, informational, and easy to navigate for users.

I created the wireframes for the landing page & contact form, leaving room to implement e-commerce if the (hypothetical) owner wanted to expand their business. The site was handcoded with HTML & vanilla CSS as a responsive mobile-first design. The logo was designed in Adobe Illustrator using a royalty-free kangaroo vector as the reference. I figured the client would want to have a simple, clean design that is easily recognizable and easy to scale for different products or media.
